# Airbnb Analytics Dashboard

## Project Overview

The Airbnb Analytics Dashboard is a data analytics project developed using Python, MySQL, and Power BI to analyze Airbnb listing data and generate meaningful business insights. The project focuses on understanding market trends, host performance, and guest experience to support data-driven decision-making in the short-term rental industry.

## Objectives

* Analyze Airbnb listings and identify market trends.
* Evaluate host performance and guest engagement.
* Understand factors affecting guest satisfaction.
* Create interactive dashboards for business analysis.
* Support strategic decision-making through data visualization.

## Technologies Used

### Programming and Data Processing

* Python
* Pandas
* NumPy

### Database Management

* MySQL

### Data Visualization

* Power BI

## Methodology

### Data Collection

The Airbnb listings dataset was imported into Google Colab for analysis and preprocessing.

### Data Cleaning

Data cleaning involved handling missing values, removing duplicate records, correcting data types, and preparing the dataset for analysis.

### Data Analysis

SQL queries were used to perform aggregations, filtering, grouping, and ranking operations to derive meaningful insights from the data.

### Dashboard Development

Power BI was used to build interactive dashboards that present key metrics and business insights in a clear and visually effective manner.

## Dashboard Structure

### Market Analysis

* Room Type Distribution
* Top Neighborhoods by Listings
* Top Neighborhoods by Reviews

### Host Analysis

* Superhost vs Non-Superhost Comparison
* Top Hosts by Number of Listings
* Review Analysis by Host Type

### Guest Experience Analysis

* Average Availability by Room Type
* Guest Satisfaction Metrics and Ratings

## Key Findings

* Entire Home/Apartment listings represent the largest share of the market.
* Certain neighborhoods generate significantly higher guest engagement and review activity.
* Superhosts generally receive greater guest interaction and visibility.
* Communication, cleanliness, and check-in experience are important contributors to guest satisfaction.
* Availability trends provide useful insights for improving occupancy and pricing strategies.

## Business Value

This dashboard enables hosts and property managers to:

* Identify high-demand locations.
* Improve guest satisfaction and service quality.
* Monitor host performance.
* Optimize availability and operational decisions.
* Support business growth through data-driven insights.
